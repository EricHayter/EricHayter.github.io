renderer2 - OpenGL Renderer

[GitHub] An OpenGL-based renderer built as an evolution of renderer(1) (see below). Features GPU-accelerated rendering using OpenGL 3.3, support for OBJ and glTF/GLB model formats, interactive camera controls, and real-time parameter tweaking with ImGui.

GoodKnight Chess Engine

[GitHub] A hybrid chess engine developed at Chess Hacks (University of Waterloo) that combines classical alpha-beta pruning with modern machine learning. The neural network was trained on millions of positions evaluated by Stockfish at depth 10. Built using Python, PyTorch, and React, and deployed via AWS Lambda.

Try playing against it here.

YADB (yet another database)

[GitHub] Yet Another Database (yadb) is a relational database that I am building from scratch using C++ to learn more about database internals.

Check out the blog if you are interested.

.obj Renderer

[GitHub] A small renderer that I built for the .obj file format commonly used in tools like Blender. Built using SDL3 and no graphics rendering APIs (such as OpenGL).

QNX Remote Process Monitor

[GitHub] Tool for developers using the QNX RTOS to remotely monitor processes running on their device while providing helpful time series data to track resource usage.
